# Apple WebSite Clone

This project is a clone of the Apple webSite front page as of November 2023 created using HTML, CSS and JS. It includes responsive design using media queries, and it utilizes Ionic icons and Images are from Apple Web Site. Please note that all product logos, designs, and trademarks belong to their respective owners, and the clone is intended for educational and non-commercial purposes only.

Getting Started

To get started with this project, you can either download the source code or clone the repository to your local machine.
Prerequisites

    A code editor (e.g., Visual Studio Code)
    A web browser

Installation

    Clone the repository using Git:

    bash

    git clone https://github.com/DilankaSandeep/apple-clone.git

    Alternatively, you can download the ZIP archive of the project and extract it to your preferred location on your computer.

    Open the project folder in your code editor.

Usage

    Open the index.html file in your web browser to view the Apple front page clone.

    Enjoy exploring the project and its responsiveness on various screen sizes.
